# Troubleshooting Tickets

This section documents realistic help desk and systems administration issues.

## Planned Tickets

1. Client cannot join domain
2. DNS resolution failure
3. User cannot access shared folder
4. RDP blocked by firewall or NSG
5. Failed login investigation

## Ticket Format

Each ticket will include:

- Problem
- Symptoms
- Root cause
- Tools used
- Fix
- Validation
