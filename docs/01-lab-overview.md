# Lab Overview

This lab is a secure mission training environment built in Microsoft Azure.

## Purpose

The goal is to demonstrate practical IT skills used in systems administration, network support, information assurance, and simulation/training support environments.

## Lab Summary

The environment uses Azure networking, segmented subnets, a Windows Server 2022 domain controller, Active Directory Domain Services, DNS, security groups, and documented security controls.

## Current Build

- Resource Group: RG-SecureMissionLab
- VNet: VNET-BlackScalpel-WE
- Region: West Europe
- Domain: blackscalpel.local
- Domain Controller: DC01
- DC01 Private IP: 10.0.2.10

## Core Components

- Azure Resource Group
