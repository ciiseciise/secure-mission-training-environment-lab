# Lab Overview

This lab documents the buildout of a secure Azure-based (blackscapel)mission training environment.

## Purpose

Demonstrate practical skills in systems administration, Azure networking, Active Directory, DNS, security controls, monitoring, and troubleshooting.

## Environment

- Microsoft Azure
- Windows Server 2022
- Active Directory Domain Services
- DNS
- Azure Virtual Network
- Azure subnets
- Future: Windows client, Linux server, Wazuh SIEM
